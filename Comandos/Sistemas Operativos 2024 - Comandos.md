También existe el comando ==awk== el cual es muy util pero es algo mas complejo de utilizar ya que permite hacer muchísimas cosas que hacen otros comandos:
- `awk '{print}' archivo` <small>(es como si hiciera cat)</small>
- `awk '/algo {print}' archivo` <small>(filtro lo que tenga la palabra "algo" y lo muestro)</small>
- `awk '{print $1,$4}' archivo` <small>(muestro las columnas 1 y 4. El 0 seria toda la linea)</small>
- `awk '{print NR,$0}' archivo` <small>(enumera las lineas y las muestra)</small>
- `awk '{print $1,$NF}' archivo` <small>(imprimo la columna 1 y la ultima columna[para eso el NF])</small>
- `awk 'NR==3, NR==6 {print NR,$0}' archivo` <small>(muestro las lineas entre 3 y 6 con todas sus columnas)</small>
- `awk '{print NR "-" $1}' archivo`  <small>(enumero la lista y separo con un - mostrando solo el primer elemento de la columna)</small>
- `awk 'NF < 0' archivo` <small>(imprimo todas las lineas no vacias)</small>
- `awk '{if(length($0) > max)max=length($0)} END {print max}' archivo` <small>(muestro la linea mas larga, los espacios cuentan como 1)</small>
- `awk 'END {print NR}' archivo` <small>(cuenta cuantas columnas tiene un archivo)</small>
- `awk 'length($0) > 10' archivo` <small>(muestro las lineas con mas de 10 caracteres)</small>
- `awk '{if($3 == "HOLA) print $0;}' archivo` <small>(encuentro una palabra en una columna especifica y muestro toda la linea)</small>
- `awk 'BEGIN {for(i=1;i<=6;i++) print "square of",i,"is",i*i;}'` <small>(da los cuadrados de los numeros 1 al 6. Imprime: square of 2 is 4)</small>
### CONECTORES DE COMANDOS

| CONECTOR                | EXPLICACION                                                                                                                       | EJEMPLO                                                                                        |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------- |
| >                       | Redirijo la salida de un comando hacia un archivo                                                                                 | echo "Hola" > archivo                                                                          |
| >>                      | Redirijo la salida de un comando hacia un archivo pero agregando una linea nueva                                                  | echo "Nueva linea" >> archivo                                                                  |
| <                       | Redirijo la entrada a un comando desde un archivo                                                                                 | cat < archivo <small>(leo lo que hay en el archivo)</small>                                    |
| \|                      | Conecto distintos comandos mediante sus entradas y salidas. Toma la salida del primero y la redirige como entrada para el segundo | ls / -l \| wc -l <small>(cuenta la cantidad de 'lineas' que hay en el directorio raiz)</small> |
| &                       | Hago que el comando se corra en modo background(bg) y pueda seguir usando la consola                                              |                                                                                                |
| && <small>(and)</small> | Ejecuta comandos en secuencia solo si el anterior se ejecuto correctamente                                                        |                                                                                                |
